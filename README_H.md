# House Renting App — REST API

A backend REST API for a house renting platform built with Spring Boot. The API handles user authentication, role-based access, and booking management.

---

## Features

- JWT-based authentication and authorization
- Role-based access control (admin and regular user roles)
- House listing management
- Booking creation and confirmation logic
- Secured endpoints tested via Postman

---

## Tech Stack

- Java, Spring Boot
- Spring Security, JWT
- REST API
- Postman (testing)

---

## How to Run

1. Clone the repo
2. Configure your database connection in `application.properties`
3. Run the project with your IDE or `./mvnw spring-boot:run`
4. Use Postman to interact with the API endpoints
